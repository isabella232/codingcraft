Keyboards
  • Move - W, A, S, D or arrow keys
  • Remove a block - Mouse click
  • Put a block - Ctrl+mouse click
  • Change the block - Tab
  • Jump - Space
  • Flying toggle - Space Space or F key
  • Go down - Shift
  • Run code - number keys
  • Exit play mode - Esc
