API
The 'cc' and 'codingcraft' is same objects. To see the result of the code, you might need to play the game
  • cc.setBlock(1, 1, 1, BLOCK.diamond);
  • cc.removeBlock(1, 1, 1);
  • cc.getBlock(1, 1, 1);
  • cc.move(1, 0, 0); // move left
  • cc.move(-1, 0, 0); // move right
  • cc.move(0, 0, 1); // move forward
  • cc.move(0, 0, -1); // move backward
  • cc.moveTo(0, 20, 0);
Examples.
for (var i = -10; i < 10; i++) {
  for (var j = -10; j < 10; j++) {
    for (var k = -10; k < 10; k++) {
      if ((i*i + j*j + k*k) < 10*10) {
        cc.setBlock(i, j+10, k, BLOCK.diamond);
      }
    }
  }
}
cc.moveTo(0, 20, 0);
